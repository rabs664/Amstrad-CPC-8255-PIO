This folder contains software associated with the Amstrad CPC 8255 PIO.

piotst04.bas is a test program written in BASIC.

piotst01.z80 is an equivalent program written in Z80 Assembley and using CPC Firmware routines

[Retro Assembler](https://enginedesigns.net/retroassembler/) was used to compile the Z80 Assembley using the [Extensions for VS Code](https://marketplace.visualstudio.com/items?itemName=EngineDesigns.retroassembler)
