This folder contains software associated with the Amstrad CPC 8255 PIO.

piotst04.bas is a test program written in BASIC.

piotst02.asm is an equivalent program written in Z80 Assembly (using RASM) and using CPC Firmware routines. piotst02.dsk is the disk created from this.

very-quick-test.txt contains some simple instructions to undertake a very quick test to see if the PIO is working.
